ProductAllocation

ProductAllocation class

An element that specifies the allocation of a single Product to a Task.

A ProductAllocation is optionally related to a DeviceElement. This is to be able to track the assignment of a single Product or of a number of different Products to a bin or to another type of DeviceElement. The transferred amount can optionally be specified by a combination of the attributes quantityDDI and quantityValue.
