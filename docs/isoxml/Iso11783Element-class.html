Iso11783Element

Iso11783Element class sealed

A sealed parent class for all ISO 11783-10 elements.

This is a starting point on which the elements extend with their attributes and children elements.

Some elements are not available in earlier versions of the standard, these have the onlyVersion4AndAbove parameter set to true, which can be used to filter them out if exporting to an older version.
