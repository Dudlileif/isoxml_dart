Worker

Worker class

An element that describes a worker that can be referenced by a Task.

All WorkerAllocations are logged with time information inside the data transfer file. The Task.responsibleWorkerIdRef has a special meaning for a Task. This Worker is directly referenced by the task without any additional logged data.
