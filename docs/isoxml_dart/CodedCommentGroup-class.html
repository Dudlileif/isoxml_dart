CodedCommentGroup

CodedCommentGroup class

An element that can be used to combine predefined CodedComments into groups.

The purpose is to have a better navigation and selection of CodedComments on the mobile system. Each CodedComment can only belong to a single group. The comments are attached to the group by adding the group's id as their CodedComment.groupIdRef.
